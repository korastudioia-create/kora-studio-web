# Kora Studio

Web oficial de Kora Studio — Agencia de IA 100% online. Webs, automatización y marketing con inteligencia artificial.

## 🚀 Stack
- HTML5 + CSS3 + JavaScript vanilla
- Sin dependencias ni build
- Hosting: Vercel
- Backend: n8n (formulario y chatbot)

## 📁 Estructura
```
kora-studio-web/
├── index.html          # Landing principal
├── privacidad.html     # Política de privacidad (RGPD)
├── aviso-legal.html    # Aviso legal (LSSI-CE)
├── cookies.html        # Política de cookies
├── robots.txt          # Crawling rules
├── sitemap.xml         # Sitemap para SEO
└── README.md
```
